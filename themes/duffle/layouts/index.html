Install and manage distributed app bundles.

Duffle is a simple command line tool that interacts with Cloud-Native Application Bundles (CNABs) - helping you package and unpackage distributed apps for deployment on whatever cloud platforms and services you use.

Get Duffle

Check out the CNAB spec
